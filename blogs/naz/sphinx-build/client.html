The SMPP client that will interact with SMSC/server.

Example declaration:

import asyncio
import naz
loop = asyncio.get_event_loop()
outboundqueue = naz.q.SimpleOutboundQueue(maxsize=1000, loop=loop)
client = naz.Client(
        async_loop=loop,
        smsc_host="127.0.0.1",
        smsc_port=2775,
        system_id="smppclient1",
        password="password",
        outboundqueue=outboundqueue,
    )

  • replace_if_present_flag (int) – Flag indicating if submitted message should replace an existing message.

  • throttle_handler – python class instance implementing functionality of what todo when naz starts getting throttled responses from SMSC
  • correlation_handler – A python class instance that naz uses to store relations between SMPP sequence numbers and user applications’ log_id’s and/or hook_metadata.
